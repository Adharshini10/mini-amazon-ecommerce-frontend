
# 📦 Mini Amazon E-Commerce Frontend

A responsive front-end e-commerce web application inspired by Amazon, built using **HTML, CSS, and JavaScript**. This project demonstrates real-world shopping features like product browsing, search & filter, product details modal, and a fully functional shopping cart using **localStorage**.

---

## 🚀 Live Demo
https://adharshini10.github.io/mini-amazon-ecommerce-frontend/

---


## ✨ Features

### 🛍️ Product Features
- Product grid layout (Amazon-style UI)
- 12+ products with images
- Category-based filtering (Electronics, Fashion, Home)
- Search functionality
- Product details popup (modal view)

### 🛒 Cart System
- Add to cart functionality
- Increase / decrease quantity
- Remove items from cart
- Cart total calculation
- Persistent cart using localStorage

### 🎨 UI/UX Features
- Responsive design (mobile + desktop)
- Modern card hover effects
- Full-screen modal overlay
- Styled navigation buttons (Home / Cart)
- Clean and minimal UI design

---

## 🛠️ Tech Stack

- HTML5 – Structure
- CSS3 – Styling & Responsive Design
- JavaScript (ES6) – Logic & Interactivity
- LocalStorage API – Cart data persistence

---

## 📁 Project Structure

mini-amazon-ecommerce-frontend/
│
├── index.html        # Home page (products list)
├── cart.html         # Shopping cart page
├── styles.css        # All styling (UI/UX)
├── products.js       # Product data
├── script.js         # Home page logic (search, filter, modal, cart)
└── cart.js           # Cart page logic

cd mini-amazon-ecommerce-frontend

### Step 3: Run Project
Open index.html in your browser

(No installation required)
